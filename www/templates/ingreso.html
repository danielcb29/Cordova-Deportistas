Bienvenido a la aplicación móvil para interacción REST con SIND COLDEPORTES

Esta aplicación es un proyecto desarrollado por Daniel Correa, estudiante de la Universidad del Valle, Colombia, para probar la comunicación interoperable bajo el protocolo REST de un servidor Django desplegado bajo el proyecto SIND COLDEPORTES y una aplicación móvil desarrollada con Apache Cordova (Angular JS)
